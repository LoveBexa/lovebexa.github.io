Data Cleaning ToolA program built in Python to prepare data ready for visualisation

This is a prototype application that combines and reformats multiple .csv files then cleans and prepares it for export to a JSON that is ready to use for visualisation. A GUI was also built to load the dataset, and translate it to JSON and then loaded back in to generate visualisations and export these.
