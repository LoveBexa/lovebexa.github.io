Coin CounterA coin counting program built in Java to sort and count coins

The task was to develop a coin sorting program using Pound sterling (£) as the default currency. The application also enables the user to check how many coins of a certain denomination can be exchanged given an input value in pennies and the denomination whilst printing the remainder.

How many coins are determined and what denominations they can exchange for while excluding a certain coin denomination with a way for user input to be validated to ensure the robustness of input/output.
